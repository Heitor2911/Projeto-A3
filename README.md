# Lista de Mangas

Uma Lista com interface onde o usuario vai poder adicionar, editar, ver e deletar itens. Onde colocara o nome do manga, a sua edição e o preço do mesmo.

# Tecnologias Utilizadas

- C#
- Windows Forms

# Funcionalidades

- Cadastro de Dados
- Validação de campo

# Objetivo
Projeto acadêmico com o intuito de testar nosso entendimento de programação orientada a objetos e conhecimentos de C#
